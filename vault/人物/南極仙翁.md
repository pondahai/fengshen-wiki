# 南極仙翁

**總出現次數**:63,**出場回數**:12 回

### 生平概述

[[南極仙翁]]為元始天尊座下弟子，長期侍奉左右，負責執行師尊各項法旨。在[[姜子牙]]奉師命下山之際，[[南極仙翁]]多次出面引導，不僅勸導其順應天數，更親自送行至麒麟崖，並交付封神榜與打神鞭等關鍵法器，同時叮囑其途中注意事項，確保西岐大業順利展開。

當[[姜子牙]]在朝歌遭遇意外導致魂魄離體時，[[南極仙翁]]正在崑崙山採芝煉藥，發現情況後立即將魂魄收入葫蘆，帶回玉虛宮向[[元始天尊]]匯報，並轉交予[[赤精子]]處理，隨後又指引[[赤精子]]前往八景宮求助，全程協助處理危機。

隨著戰事推進至五關之後，[[南極仙翁]]參與了多場重大陣戰。他曾赴瑤池向西王母借得聚仙旗以助破敵；在紅沙陣中親手擊退敵人並救出周武王；最終在西岐大軍迎戰截教十絕陣及三霄娘娘期間，[[南南極仙翁]]隨[[太上老君]]等人進陣，依令收服多寶道人座下騎獸，展現其在伐滄行動中的重要角色。

### 重要事蹟

*   **勸導與送行**：奉[[元始天尊]]之命勸導[[姜子牙]]順應天數下山，送至麒麟崖並吩咐前途保重（第15回）。
*   **交付封神榜**：奉[[元始天尊]]之命取出封神榜交給[[姜子牙]]（第37回）。
*   **警告與護法**：送[[姜子牙]]出宮並警告其不可應人呼喚；見[[申公豹]]使用幻術迷惑[[姜子牙]]時，命令白鶴童子化鶴銜走[[申公豹]]的頭顱（第37回）。
*   **寬恕申公豹**：解釋[[申公豹]]幻術的原理，並因[[姜子牙]]求情，命白鶴童子放下[[申公豹]]頭部使其復生，隨後斥責[[申公豹]]左道惑人並將之趕走（第37回）。
*   **交付打神鞭**：奉[[元始天尊]]之命取出打神鞭（第38回）；再次送[[姜子牙]]至麒麟崖（第38回）。
*   **拾獲魂魄**：在崑崙山下遇見[[姜子牙]]魂魄，將其裝入葫蘆帶往玉虛宮，並向[[元始天尊]]細細陳說經過（第44回）。
*   **轉介救援**：將裝有魂魄的葫蘆交給[[赤精子]]，領受[[元始天尊]]旨意後告知[[赤精子]]去八景宮見[[太上老君]]，並在玉虛宮外詢問[[赤精子]]是否成功救回魂魄（第44回）。
*   **隨侍進陣**：執羽扇跟隨[[元始天尊]]，並受命收拾香輦，準備進入黃河陣（第50回）。
*   **破紅沙陣**：打點破紅沙陣，同白鶴童兒至陣前挑戰；進入陣內用五火七翎扇搧退張紹撒下的紅沙（第51回）。
*   **救助武王**：破陣後發雷驚動[[哪吒]]與[[雷震子]]，發現武王已死，命[[雷震子]]背負武王屍骸放置篷下沐浴（第51回）。
*   **借取聚仙旗**：奉命前往瑤池，向西王母奏請借得聚仙旗，回到西岐後將旗交給[[姜子牙]]以幫助破解殷郊（第65回）。
*   **餞別**：向[[姜子牙]]奉酒餞別（第67回）。
*   **質問破陣時機**：詢問[[元始天尊]]既然已經入陣，今日為何不破陣以幫助[[姜子牙]]東行（第77回）。
*   **迎鑾接駕**：持玉符先行，跨鶴而來；被馬遂阻擋去路後，因知掌教師尊將至而按落雲頭回歸本陣；率眾仙迎鑾接駕，上篷坐下（第82回）。
*   **降服騎獸**：領命至篷下將虯首仙打出原身；領令將靈牙仙現出原身；遵[[太上老君]]命令將金光仙頸上連拍幾下使其現原形；一一如命施為，將名牌掛於獅、象、犼頸上（第83回）。

### 人物關係

*   **師徒**：[[元始天尊]]（師父）、[[姜子牙]]（徒弟/奉師命引導對象）、[[赤精子]]（師弟/交接魂魄物件者）、[[白鶴童子]]（門下/執行任務者）
*   **盟友**：[[太上老君]]（共同進陣合作）、[[哪吒]]、[[雷震子]]（戰場協力者）
*   **對手**：[[申公豹]]（曾施幻術迷惑姜子牙而被懲罰）、[[張紹]]（紅沙陣中對立者）、[[虯首仙]]、[[靈牙仙]]、[[金光仙]]（進陣中被降服的對手）
