# 鄂崇禹

**別名**:南伯侯

**總出現次數**:56,**出場回數**:23 回

### 生平概述

[[鄂崇禹]]為商朝四大伯侯之一的[[鄂崇禹|南伯侯]]，曾率領八百鎮諸侯朝覲於商，並在顯慶殿接受[[紂王]]慰勞及叩頭謝恩[[第2回]]。後因[[紂王]]詔令至朝歌，與其他三位伯侯會齊[[第10回]]。

在朝期間，鄂崇禹性格剛直，曾在酒席上當面指責[[崇侯虎]]過惡多端、剝民利己，並以酒壺劈面打向對方，幸得[[姜桓楚]]架開[[第10回]]。隨後更與[[姬昌]]、[[崇侯虎]]一同出班為[[姜桓楚]]求情，並上疏諫諍，請求[[紂王]]退奸除佞[[第10回]][[第11回]]。

然而，[[紂王]]以「叱主之愆」定罪於他，鄂崇禹最終被處以梟首之刑，其身死後還遭到[[妲己]]陷害，被下令碎醢其屍，導致一身首異處[[第11回]][[第95回]]。

### 重要事蹟

*   作為[[鄂崇禹|南伯侯]]，率領八百鎮諸侯朝覲於商[[第2回]]。
*   在顯慶殿接受[[紂王]]慰勞並叩頭謝恩[[第2回]]。
*   被[[紂王]]詔令至朝歌，與[[姜桓楚]]、[[姬昌]]、[[崇侯虎]]會齊[[第10回]]。
*   在酒席上當面指責[[崇侯虎]]過惡多端、剝民利己[[第10回]]。
*   以酒壺劈面打向[[崇侯虎]]，被[[姜桓楚]]架開[[第10回]]。
*   在早朝時與[[姬昌]]、[[崇侯虎]]一同出班為[[姜桓楚]]求情[[第10回]]。
*   與[[姬昌]]、[[崇侯虎]]等人上疏諫諍，請求[[紂王]]退奸除佞[[第11回]]。
*   被[[紂王]]以叱主之愆定罪，最終被梟首[[第11回]]。
*   被[[紂王]]聽信[[妲己]]陰謀，不分皂白，碎醢其屍，一身首異處[[第95回]]。
*   被[[紂王]]指控縱惡為叛，已伏誅[[第27回]]。
*   具文書知會欲伐成湯[[第67回]]。

### 人物關係

*   **君主**：[[紂王]]
*   **政敵/衝突對象**：[[崇侯虎]]（鄂崇禹曾當面指責其過惡並攻擊之）
*   **盟友/同僚**：[[姜桓楚]]、[[姬昌]]（三人曾共同為姜桓楚求情及上疏諫諍；鄂崇禹亦曾協助姜桓楚擋下對崇侯虎的攻擊）
*   **支持/辯護者**：[[比干]]、[[楊任]]（二人曾為鄂崇禹辯護，稱其勤勞王家且直諫聖聰）
