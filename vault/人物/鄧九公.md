# 鄧九公

**總出現次數**:198,**出場回數**:23 回

### 生平概述

[[鄧九公]]原為商朝鎮守三山關的大將，長期與南伯侯[[鄂崇禹]]（原文作鄂順，依列表正名為鄂崇禹）的叛軍交戰七載，致使庫藏空虛。後被[[聞仲]]調遣，最終受[[紂王]]任命為主帥討伐西岐。初期他倚仗[[土行孫]]法力，曾許配女兒[[鄧嬋玉]]予其，但因[[土行孫]]逃離且被西岐收服，[[鄧九公]]陷入困境。經[[散宜生]]議親及[[太鸞]]獻計失敗後，見[[鄧嬋玉]]歸來勸降，遂棄暗投明，歸順西周。

歸周後，[[鄧九公]]成為姜子牙麾下重要戰將，屢立戰功。他在陣前斬殺劉甫、錢保、馬善、柏顯忠等將領，並擒獲鄭倫，展現了出色的武藝。然而，在攻打青龍關時，[[鄧九公]]主動請纓挑戰陳奇，因不熟悉陳奇的黃氣法術，戰三十回合後遭敵方異術擊中，魂魄散失，最終被擒並斬首示眾，陣亡於西岐軍中。

### 重要事蹟

*   **三山關歷戰**：在南伯侯[[鄂崇禹]]攻取三山關期間苦戰多年，導致庫藏空虛；與[[鄂崇禹]]叛軍在大戰七載中殺戮相半。（第18、20、35回）
*   **統兵伐西岐**：被[[聞仲]]發火牌調往麾下，隨後被[[紂王]]任命為征討西岐主帥，賜自旄黃鉞，得專征伐。（第51、53回）
*   **初戰折損**：率軍至西岐東門安營，派遣[[太鸞]]擊敗[[南宮适]]；自身與[[黃飛虎]]大戰勢均力敵，後被[[哪吒]]以乾坤圈打中左臂，敗回大營。（第53回）
*   **許婚土行孫**：因手臂疼痛度日如年，囑咐女兒[[鄧嬋玉]]報仇；接受[[土行孫]]金丹治癒傷勢後，見其連勝並擒將，深信其言，許配女兒給[[土行孫]]。（第53、54回）
*   **誤信刺客**：批准[[土行孫]]當晚進西岐行刺[[姬發]]與[[姜子牙]]，見其遁走後以為天子洪福，聽信謊稱未能下手，對行刺失敗毫不知情。（第54回）
*   **議親與反間計失敗**：[[土行孫]]被俘後，[[散宜生]]前來議親；[[鄧九公]]起初拒絕，後聽從[[太鸞]]計謀假意答應，設伏欲擒[[姜子牙]]，卻因信號砲響措手不及，營壘大亂而敗退。（第55、56回）
*   **棄暗投明**：追殺五十餘里後收集敗殘人馬，因失去女兒[[鄧嬋玉]]傷感，見其歸來勸降後，最終決定歸順西周，與[[姜子牙]]並轡敘話，次日朝賀[[姬發]]。（第56回）
*   **斬將擒敵**：
    *   奉命率左營進攻，一刀砍死[[劉甫]]。（第61回）
    *   從旁用刀蓋住[[鄭倫]]的降魔杵，將其擒獲。（第61回）
    *   在陣前與[[錢保]]大戰三十回合，回馬刀將其劈於馬下並梟首。（第62回）
    *   出城與[[馬善]]交戰十二三回合，生擒[[馬善]]。（第63回）
    *   奉命沖殺左糧道門，一刀劈死[[張山]]，隨即圍困[[殷郊]]。（第65回）
    *   大戰[[柏顯忠]]二三十回合後，賣個破綻將其揮於馬下。（第66回）
    *   主動請纓，大戰三十回合後將[[馬方]]劈於馬下並梟首。（第73回）
    *   在戰場上奮勇走馬，刀劈[[余成]]；一刀將[[孫寶]]揮於馬下並梟首。（第73回）
*   **陣亡青龍關**：出戰[[陳奇]]，大戰三十回合後因[[陳奇]]噴出黃氣導致魂魄散失而落馬被擒；醒後大罵[[陳奇]]，隨後被[[丘引]]下令推出斬首，首級號令於關上。（第73、74回）

### 人物關係

*   **親屬**：[[鄧嬋玉]]（女兒）。
*   **師徒/主僕**：[[土行孫]]（部下、曾許配女兒者）。
*   **盟友**：[[姜子牙]]（歸周後的君主/上司）、[[姬發]]（君王）、[[黃飛虎]]（同僚，曾任命其為先行）。
*   **對手**：[[鄂崇禹]]（長期交戰對象）、[[哪吒]]（打傷其左臂）、[[鄭倫]]（被擒）、[[陳奇]]（致其陣亡者）、[[丘引]]（下令斬首者）、[[張山]]（被殺）、[[柏顯忠]]（被殺）、[[馬方]]（被殺）、[[余成]]（被殺）、[[孫寶]]（被殺）、[[錢保]]（被殺）、[[劉甫]]（被殺）、[[馬善]]（被擒）。
